# ROADMAP

## 功能

- 确认当前插件的开发进度，接下来需要稳固插件的功能了；还有因为会启动一个监听端口让 Openclaw 建立通信，那个端口的健康状态应该在 UI 中显示出来，不应该藏着设置里面
- telegram 支持当 openclaw 在处理消息时，提醒用户 typing ，当前不支持
- 支持国际化 i18n ，优先提供中英文两种，支持设置中切换；同时做一次项目代码清洗去掉所有代码文件中的中文

## 缺陷

- 当前 openclaw 没有能成功通过 channel 把图片发过来
- 进行中的对话应该在会话列表中有个小标记才对，特别是最近有新消息需要查看
- 会话搜索的功能貌似不可用的
- 文件浏览器没有名称搜索的能力，现在的几个下拉比较原始，看下 @openclaw-desktop-design.md 的文件浏览器设计
